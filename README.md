# 📊 DART 기업분석 대시보드

한국 금융감독원 DART API를 활용한 실시간 기업 재무 분석 대시보드입니다.

## 🚀 기능
- 실시간 기업 재무 데이터 조회
- AI 기반 재무 분석 (GPT-4, Gemini)
- 실시간 웹 검색 연동
- 데이터베이스 저장 및 관리
- 시각화 차트

## 🔧 사용법
1. 사이드바에서 기업 검색
2. 분석하고 싶은 연도와 보고서 유형 선택
3. AI 분석탭에서 질문 입력

## 🔑 API 키 설정
- OpenAI API 키: GPT 분석용
- Google AI API 키: Gemini 분석용
